About Presets

Screening presets contain pre-configured screening types for different medical specialties. These are standardized configurations that can be imported to quickly set up common screening protocols.

Available Preset Categories:
  • Primary Care
  • Cardiology
  • Women's Health
  • Oncology
  • Endocrinology
Importing presets will create new screening types in the system. Check "Overwrite existing" to update screening types with the same name.
