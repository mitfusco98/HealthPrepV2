What happened?

The server encountered an unexpected condition that prevented it from fulfilling your request.

This is usually temporary and caused by:
  • High server load or temporary resource limitations
  • Database connectivity issues
  • Third-party service interruptions (OCR, FHIR)
  • System maintenance or updates in progress
Troubleshooting Steps
  1. Wait and retry: Most 500 errors are temporary
  2. Refresh the page: Simple browser refresh often resolves the issue
  3. Clear browser cache: Outdated cache can cause conflicts
  4. Try a different browser: Rule out browser-specific issues
  5. Check system status: Verify if maintenance is in progress
