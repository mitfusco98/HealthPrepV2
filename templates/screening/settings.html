Best Practices
Cutoff Logic Hierarchy:
  1. 1 Data Type Cutoffs: Individual values for each medical data type
  2. 2 "To Last Appointment": When cutoff = 0, uses patient's last appointment date
  3. 3 Default Fallback: 6 months if no past appointments found
Best Practices:
  • Lab results: 6-12 months for chronic conditions
  • Imaging: 12-36 months depending on study type
  • Consults: 6-12 months for relevance
  • Hospital: 12 months for recent admissions
Tip: Set cutoff to 0 for patient-specific, contextual filtering based on their last appointment.
